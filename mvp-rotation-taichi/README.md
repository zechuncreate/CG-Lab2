# 3D MVP Transformation Based on Taichi

## 1. 项目概述

本项目基于 Taichi 编程框架，实现了计算机图形学中经典的三维几何变换流程——MVP（Model-View-Projection）变换。通过对三维空间中几何体（立方体）的顶点进行矩阵变换，并映射至二维屏幕空间，实现了一个可交互的三维线框立方体渲染系统。

该项目完整复现了从模型空间到屏幕空间的标准图形学管线，具有明确的数学结构与工程实现对应关系。

---

## 2. 基本原理

在三维图形渲染过程中，顶点需依次经过以下变换：

Model → View → Projection → NDC → Screen

对应矩阵形式为：

MVP = Projection × View × Model

* **Model（模型变换）**：对物体进行旋转变换（支持 X/Y/Z 三轴）
* **View（视图变换）**：将相机位置变换至原点
* **Projection（投影变换）**：将三维视锥体映射至标准立方体（NDC）

在完成齐次坐标变换后，通过透视除法（Perspective Divide）将坐标归一化至 ([-1, 1]) 区间，再映射至屏幕坐标系。

## 4. 功能实现

本项目实现了以下核心功能：

* 三维立方体几何建模（8个顶点，12条边）
* 基于齐次坐标的矩阵变换体系
* 模型变换（支持多轴旋转）
* 视图变换（相机坐标变换）
* 透视投影（Perspective Projection）
* MVP 矩阵组合计算
* 透视除法（Perspective Divide）
* 标准设备坐标（NDC）到屏幕空间映射
* 基于 Taichi GUI 的实时线框渲染
* 键盘交互控制三维旋转

## 8. 技术要点分析

本项目的关键技术点包括：

1. **齐次坐标系统**

   * 使用四维向量统一表示平移与线性变换

2. **矩阵组合顺序**

   * 采用列向量约定，变换顺序为右乘：MVP = P × V × M

3. **透视投影构造**

   * 通过“透视 → 正交”两步构造投影矩阵

4. **透视除法**

   * 将裁剪空间坐标转换为标准设备坐标

5. **三维空间感的实现**

   * 通过 Y 轴旋转与透视投影共同体现空间深度

## 9. 总结

本实验通过构建完整的 MVP 变换流程，实现了从理论推导到工程实现的统一。通过对三维立方体的实时渲染与交互控制，直观验证了矩阵变换在计算机图形学中的核心作用。

该项目不仅加深了对三维坐标变换的理解，也为后续学习光照模型、深度测试等高级图形技术奠定了基础。
